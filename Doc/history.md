# 更新履歴

* Ver.3.0.3

    * Unicodeファイル名の操作に対応
    * VCランタイム未インストールで起動したとき原因不明のエラーになるためメッセージを修正
    * 一部のダイアログでレイアウトが崩れていたため修正

* Ver.3.0.2

    * 高解像度モニターへの対応
    * ファイルビューアのビジュアルベルが遅い問題を調整
    * バグ修正

        * グラフィックビューアの画像フィルターが正しく動作していなかった問題を修正
        * Windowsのホスト名に「-」が入っているとルートフォルダの表示でクラッシュするバグを修正
        * グラフィックビューアの白背景/黒背景のトグルで、1回目の切り替えが正しく動作しない不具合を修正
        * ZIPフォルダのファイル数が減る方向にディレクトリ変更するとクラッシュするバグを修正

* Ver.3.0.1

    * キー定義変更時にエラーが発生する不具合を修正（セットアップのコマンド一覧の設定ファイルを修正）

* Ver.3.0.0

    * オープンソース版初回リリース
